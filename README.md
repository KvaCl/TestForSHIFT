#  Booking System

Сервис бронирования переговорных комнат с поддержкой ролей (сотрудник/администратор), реализованный на FastAPI с соблюдением REST-принципов.

##  Возможности

-  **Две роли пользователей**: сотрудники и администраторы
-  **Управление комнатами**: создание, просмотр, удаление (только для админов)
-  **Временные слоты**: каждая комната имеет заранее заданные слоты
-  **Проверка доступности**: просмотр свободных слотов на конкретную дату
-  **Бронирование**: создание и отмена броней
-  **Защита данных**: сотрудники видят только свои брони, админы — все
-  **JWT-аутентификация** с ограниченным сроком действия (30 минут)
-  **REST API** с автоматической документацией Swagger

## Установка и запуск

### Вариант 1: Через Docker Compose 

```bash
# Клонировать репозиторий
git clone https://github.com/KvaCl/TestForSHIFT
cd TestForSHIFT

# Запустить приложение + PostgreSQL
docker-compose up --build

# Остановить
docker-compose down

```
После запуска откройте в браузере:
- **Swagger UI**: http://127.0.0.1:8000/docs
- **ReDoc**: http://127.0.0.1:8000/redoc

### Вариант 2: Локальный запуск

Требует установленной PostgreSQL на компьютере.

```bash
# Клонировать репозиторий
git clone https://github.com/KvaCl/TestForSHIFT
cd TestForSHIFT

# Установить Poetry (если не установлен)
pip install poetry

# Установить зависимости
poetry install

# Настроить подключение к БД в app/database.py:
# DATABASE_URL = "postgresql+psycopg://postgres:admin@127.0.0.1:5432/postgres"

# Запустить сервер
poetry run uvicorn app.main:app --reload
```

Документация API: http://127.0.0.1:8000/docs
